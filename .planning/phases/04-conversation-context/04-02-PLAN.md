**1. Get bot_user_id available to handlers:**

In `app.py`, after `app = App(...)`, add:
```python
auth_response = app.client.auth_test()
BOT_USER_ID = auth_response["user_id"]
```
Store as module-level. Pass to register functions or make accessible via import. Simplest: add `BOT_USER_ID` to app.py and import it in handlers. Alternatively, each handler can call `client.auth_test()["user_id"]` but that's wasteful. Use the app.py approach — import `BOT_USER_ID` from `app` module. BUT beware circular imports since handlers are imported in app.py.

Better approach: Cache bot_user_id lazily in `handlers/shared.py`:
```python
_bot_user_id: str | None = None

def get_bot_user_id(client) -> str:
    global _bot_user_id
    if _bot_user_id is None:
        _bot_user_id = client.auth_test()["user_id"]
    return _bot_user_id
```
